To run this code locally:

	Install all files to a new folder, strictly maintaining the directory structure in GitHub

	Open the folder in VScode

	In a new VScode terminal, run the following:

		npm install express cors node-fetch dotenv

	On line 6 of server.js, replace “const OPENAI_API_KEY = process.env.OPENAI_API_KEY;” with:
		 “const OPENAI_API_KEY = <your_openai_api_key>;

	Run the following command: 
		node server.js

	With the server now running, go to:
		http://localhost:3000
